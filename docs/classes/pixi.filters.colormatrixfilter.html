    • Defined in definitions/lunalite-pixi-mv.d.ts:1892

    The ColorMatrixFilter class lets you apply a 5x5 matrix transformation on the RGBA color and alpha values of every pixel on your displayObject to produce a result with a new set of RGBA color and alpha values. It's pretty powerful!

     var colorMatrix = new ColorMatrixFilter();
 container.filters = [colorMatrix];
 colorMatrix.contrast(2);
